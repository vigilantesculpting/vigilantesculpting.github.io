Stormblade assemble!

Published on 22/04/2021 @11:53:00 by vigilante sculpting

Here is a mockup of the current state of the Stormblade. I have one length of track assembled, and the wheelbase is ready, but it still needs to be attached to the hull.I want to make it fairly stable, but I haven't decided on exactly how to do it yet.

Here is the hull panel removed, so you can see the track running through the hull.If I can get the track to run smoothly, I will consider the next step which would be to try and add motors. :)

Assembling the tracks is pretty mind numbing in a very zen sort of way, if you like that kind of thing.

I added rubber strips to the roadwheels, so they grip the track a bit better. Resin-on-resin friction is almost non-existant.
