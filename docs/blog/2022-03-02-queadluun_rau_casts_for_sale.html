I've put a couple of resin casts of my Queadluun Rau model up for sale on etsy.There is a limited number of casts available, since I don't know what the demand for this might be.(If there are no more left, please get in contact with me and we'll make it happen!)

Basically, what you get is a 70mm model that is cast in Alumilite Slow Set 7 polyurethane resin.It needs to be assembled and painted.

The model is designed in a flying pose, so it should be mounted on a suitable base.I don't provide a base, this is where you get to be creative!
